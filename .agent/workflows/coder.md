# /coder Workflow

This workflow activates DEV.Coder to execute technical tasks based on validated plans and architecture.

## Steps

1. **Load Persona**
   Read `Lω_Agentic/Personas/DEV.Coder.md`

2. **Load Rules**
   Read `Lω_Agentic/Rules/DEV.Coder_Rules.md`

3. **Load Context**
   Read mandatory context:
   - `L2_Design/.../Architecture.md` (Design specs)
   - `L2_Design/.../Shared_Standards.md` (Coding standards)
   - `L3_Technical/Plans/.../[Plan]_*.md` (Active plan)

4. **Execute**
   - Translate Plan Tasks into Code actions.
   - Follow strict Naming Conventions.
   - Update status in Task Tracker (if active).

## Constraints
- Must stick to L3 Plan scope.
- Must follow L2 Architecture patterns.
- All output in French (interaction) / English (Code/Docs).
